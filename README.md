# rMFilter
Acceleration of long read-based structure variation detection with chimeric read filtering

###Getting Start
	git clone https://github.com/hitbc/LAMSA.git
	cd LAMSA; make
	./lamsa index ref.fa
	./lamsa aln ref.fa read.fq > aln.sam
